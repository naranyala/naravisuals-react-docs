# Introduction to C# and Avalonia UI

Welcome to the guide for building modern, cross-platform desktop applications using **C#** and **Avalonia UI**.

## What is C#?
C# is a powerful, type-safe, object-oriented programming language developed by Microsoft.

## What is Avalonia UI?
**Avalonia UI** is a modern, open-source, cross-platform UI framework for .NET.
