# Getting Started with Avalonia UI

## Prerequisites
1. **.NET SDK**: Install from [dotnet.microsoft.com](https://dotnet.microsoft.com/download).
2. **IDE**: JetBrains Rider or Visual Studio.

## Setup
```bash:desc=Install Avalonia templates
dotnet new install Avalonia.Templates
```

```bash:desc=Create a new project
dotnet new avalonia.app -o MyFirstAvaloniaApp
```
